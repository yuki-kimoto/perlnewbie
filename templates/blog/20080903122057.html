uc function - Convert lowercase string to uppercase string

uc function convert lowercase string to uppercase string. Each character is converted from lowercase to uppercase if possible. 
my $str = uc "aaa";

uc function programing example

uc function programing example. 
use strict;
use warnings;

# Convert lowercase "aaa" to uppercase "AAA"
my $str = uc "aaa";

print "$str\n";
Output 
AAA

uc function relative information

  • lc function - Convert uppercase to lowercase
  • lcfirst function - Convert uppercase to lowercase
  • ucfirst function - Convert first character of string to uppercase
