lcfirst function - Convert first character of string to lowercase

lcfirst function convert character of string to lowercase. 
# Convert first character of string to lowercase
$ret = lcfirst $string;

lcfirst function example program

This is lcfirst function example program. 
use strict;
use warnings;

# Convert "FOO" to "fOO"
my $str = lcfirst "FOO";

print "$str\n";
Output: 
fOO
